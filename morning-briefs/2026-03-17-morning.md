# 2026年3月17日 早报

## 1. 今日最值得知道的事

OpenClaw 项目昨晚至今有重要动态：1）发布了 v2026.3.13-1 版本更新（3月14日），你当前运行的正是此版本；2）GitHub 上出现了几个关键 bug 报告，包括模型路由错配问题（#48574）和嵌入式会话状态泄漏（#48573），这些对 Mac mini 本地部署的稳定性有直接影响；3）项目 star 数已达 31.7 万，社区活跃度持续增长。建议关注模型路由问题的修复进展。

## 2. 今天值得试的一个 skill / 工具

推荐 **openclaw-github-assistant** 技能（ClawHub 评分 3.658）。这个技能专门为 OpenClaw 用户设计，能直接在 Telegram 中管理 GitHub issues、PRs 和 CI 状态。对于使用 Mac mini + Telegram 工作流的你，这个技能能减少上下文切换，直接在聊天中处理 GitHub 任务，提升开发效率。安装后可通过简单命令查看项目动态和 CI 状态。

## 3. 值得关注的一个项目 / 更新

**openclaw-control-center**（GitHub 新星项目，3月11日创建，已有 1882 star）。这个项目将 OpenClaw 从黑盒转变为可视化控制中心，提供实时监控、配置管理和性能分析界面。对于重视稳定性和可控性的你，这个工具能让你更直观地了解 OpenClaw 运行状态，及时发现潜在问题，特别适合 Mac 本地部署的调试和维护。

## 4. 一句给朽木的实用建议

建议今天检查 OpenClaw 的模型路由配置，确保 provider 和 model ID 正确匹配（参考 issue #48574），避免因配置错配导致的稳定性问题。同时可以关注 openclaw-control-center 项目，考虑是否适合你的本地监控需求。
