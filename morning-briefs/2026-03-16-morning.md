# 2026年3月16日 早报

## 1. 今日最值得知道的事

OpenClaw 社区昨晚至今有多个重要修复和功能更新：1）修复了 fallback model 永久覆盖 agent 配置的问题（#47705），这对 Mac mini 本地部署的稳定性很重要；2）新增了 /trust 和 /untrust 命令用于临时提升 exec 安全权限（PR #47696），在需要临时执行敏感命令时更安全可控；3）修复了 logfile rotation 的文件路径解析时机问题（PR #47700），避免日志文件管理异常。这些更新都针对稳定性和安全性改进。

## 2. 今天值得试的一个 skill / 工具

推荐关注 **changelog-generator** 技能（PR #47698 新增）。这个技能能自动从 git 提交历史生成规范的 changelog，特别适合维护 OpenClaw 技能或开源项目。对于重视实用工作流的你，这个技能可以简化版本发布流程，确保变更记录清晰可追溯。安装后可通过简单命令生成专业级的更新日志。

## 3. 值得关注的一个项目 / 更新

**n8n**（工作流自动化平台）持续更新中。这个开源项目特别适合个人助理场景，可以连接 Telegram、GitHub、OpenAI 等数百种服务，实现复杂的自动化工作流。对于使用 Mac mini 本地 OpenClaw 的你，n8n 提供了可视化编排界面，能扩展 OpenClaw 的自动化能力，比纯代码方案更易维护和调试。

## 4. 一句给朽木的实用建议

建议今天检查一下 OpenClaw 的 fallback model 配置是否正常（如果遇到模型切换问题），并试用新的 /trust 命令来管理临时权限提升，这样在执行需要批准的命令时会更安全可控，符合你重视稳定性的偏好。
